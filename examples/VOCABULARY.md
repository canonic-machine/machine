## Example Terms

### example
A self-contained demonstration of FSM concepts. Shows how to apply CANONIC governance to a specific use case.

### minimal example
The simplest possible implementation of the 4-state FSM. Demonstrates core structure without complexity.

### self-contained
An example that includes all necessary files and can be validated independently without external dependencies.
