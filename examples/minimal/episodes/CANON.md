# CANON (Episodes)

**Governance for episode state.**

**Inherits from:** [../../CANON.md](../../CANON.md)

---

## Constraints

### File naming

Episodes must follow format: `NNN-*.md` where NNN is zero-padded sequential number.

**Violation:** Episode file does not follow NNN-*.md naming format

### Content

Episodes are ungoverned. Any format, any content, contradictions allowed.

**Violation:** None (episodes are explicitly ungoverned)

### Immutability

Episodes become immutable after asset extraction.

**Violation:** Episode modified after asset extraction

End episodes CANON.
