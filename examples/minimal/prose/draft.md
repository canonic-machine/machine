# User API Documentation

A simple REST API for managing users.

## Endpoints

### Create User

**asset-0001** creates a new **asset-0006** in the system.

The request must include **asset-0007** and **asset-0008** fields. The **asset-0008** field will be validated using **asset-0010**.

The system automatically sets the **asset-0009** timestamp.

### List Users

**asset-0002** returns all users in the system.

### Get User

**asset-0003** retrieves a single **asset-0006** by their unique identifier.

### Update User

**asset-0004** modifies an existing **asset-0006**. You can update the **asset-0007** and **asset-0008** fields.

### Delete User

**asset-0005** removes a **asset-0006** from the system.

## Data Model

The **asset-0006** entity contains:
- **asset-0007**: The user's name
- **asset-0008**: The user's email address (validated by **asset-0010**)
- **asset-0009**: When the user was created
